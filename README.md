# Diego Pulido Rivero — Immersive Developer Portfolio

An immersive sci-fi inspired developer portfolio built with Astro, React and Three.js.

Designed as an interactive command hub rather than a traditional portfolio, this project combines frontend engineering, cinematic UI design and real-time 3D rendering to create a premium navigation experience focused on performance, atmosphere and visual identity.

---

## Live Demo

https://portfolio.puvero.es

---

# Overview

This portfolio was conceived as a fully immersive digital environment instead of a conventional developer website.

The experience is inspired by futuristic spacecraft interfaces, holographic systems and cinematic UI design, transforming navigation into an interactive command center where each section behaves as part of a larger spatial interface.

The project combines:
- modern frontend architecture
- real-time 3D rendering
- immersive motion systems
- advanced UI composition
- performance-focused rendering techniques

while maintaining a clean and scalable codebase.

# Architecture

The project is built around a HUB-based navigation system where the user interacts with a central holographic interface.

The visual hierarchy is divided into:
- central command hub
- holographic navigation nodes
- immersive background systems
- modular section panels
- cinematic overlays and transitions

Special attention was given to:
- GPU-friendly rendering
- animation smoothness
- scalable component structure
- immersive UI consistency
- low visual latency

# Local Development

Clone the repository:

```bash
git clone https://github.com/DiegoPulido33/Portfolio.git
```

Install dependencies:

```bash
npm install
```

Run development server:

```bash
npm run dev
```

# Deployment

The application is deployed on Vercel using Astro server output mode and serverless API routes.

Optimized for:
- fast global deployment
- server-side API execution
- modern edge-compatible architecture

# Design Philosophy

The objective of this project was to explore the intersection between:
- software engineering
- cinematic interface design
- immersive digital experiences
- real-time frontend rendering

Instead of presenting projects through static sections, the portfolio itself becomes the product experience.

The interface was intentionally designed to feel:
- spatial
- reactive
- holographic
- cinematic
- premium
- interactive

while preserving usability and performance standards expected from production-grade frontend applications.
